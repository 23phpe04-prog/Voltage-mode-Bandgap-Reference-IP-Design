# Voltage Mode Bandgap Reference IP Design
**Sky130 PDK | ngspice | Magic | Netgen**
This project presents the design and simulation of a Bandgap Reference (BGR) circuit using open-source EDA tools. The BGR generates a temperature-independent reference voltage suitable for low-power analog systems such as LDO regulators and ADCs. The design is implemented and verified using ngspice, magic, and netgen tools
